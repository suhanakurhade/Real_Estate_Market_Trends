# Real Estate Market Trends Dashboard

## Overview

This project is an interactive Power BI dashboard developed as part of the CodeAlpha Power BI Internship Program. The dashboard analyzes property prices, housing characteristics, and market trends to support real estate investment and decision-making.

## Objectives

* Analyze property prices across different locations.
* Compare housing characteristics such as area, bedrooms, and garage availability.
* Identify market trends based on property conditions.
* Provide interactive visualizations for better decision-making.

## Dataset

The dataset contains the following attributes:

* Property ID
* Area
* Bedrooms
* Bathrooms
* Floors
* Year Built
* Location
* Condition
* Garage Availability
* Price

## Dashboard Features

### KPI Cards

* Total Properties
* Average Property Price
* Average Area
* Average Bedrooms

### Visualizations

* Property Price by Location
* Property Condition Distribution
* Area vs Price Analysis
* Bedrooms vs Price Analysis
* Garage Impact on Property Price

### Interactive Filters

* Location
* Condition
* Garage
* Bedrooms

## Tools Used

* Power BI Desktop
* CSV Dataset
* Data Visualization
* Interactive Slicers and Filters

## Key Insights

* Property prices vary significantly across locations.
* Larger properties generally command higher prices.
* Garage availability can influence property value.
* Property condition affects market pricing trends.
* Bedroom count impacts average property prices.
